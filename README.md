# 🚀 API SubLedger - Backend & Docker Environment

Ce projet est une API REST développée avec **Node.js**, **Express**, et **Mongoose (MongoDB)**, entièrement conteneurisée avec **Docker** et testée avec **Jest** et **Supertest**.

---

## 🛠️ Technologies Utilisées

* **Backend :** Node.js, Express.js
* **Base de données :** MongoDB
* **Conteneurisation :** Docker, Docker Compose
* **Tests :** Jest, Supertest
* **Validation & Sécurité :** Bcrypt, Dotenv

---

## 📦 Configuration et Lancement avec Docker

Pour lancer l'environnement de développement complet (Serveur API + Base de données MongoDB), exécutez la commande suivante :

```bash
docker compose up --build
